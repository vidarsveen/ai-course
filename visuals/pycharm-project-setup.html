üìÅ Folder & File Organization
.venv/ (Virtual Environment)
Isolates your project's Python packages from system Python. PyCharm manages this automatically.
src/ (Source Code)
All Python code goes here. main.py is entry point. Other .py files contain logic, utils, etc.
tests/ (Unit Tests)
Automated tests for your code. PyCharm can run these directly.
.git/ (Version Control)
Stores Git history. PyCharm has integrated Git tools in the VCS menu.
templates.json / requirements.txt / README.md
Project data, dependencies, and documentation. Use for context with AI assistants.
