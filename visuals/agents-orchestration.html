Key Agent Concepts

Planning (Decomposition)
Breaking a large goal into smaller, actionable steps. Good planning is criticalâ€”it's often the biggest determinant of agent success. The agent might create an explicit task list or plan one step at a time using ReAct loops.
Tool Invocation & Function Calling
The agent decides when to use a tool and formats the request correctly. Function calling (in APIs like OpenAI) provides strict validation of tool calls, making agents more robust than relying on JSON parsing.
Tool Orchestration
Managing complex, multi-step tool execution. This includes handling retries, managing state, and sequencing tool calls. It's what allows agents to be resilient to failures in the real world.
Autonomy & Human-in-the-Loop
Autonomous agents can operate over long planning horizons with minimal intervention. However, critical decisions often require a "human-in-the-loop" checkpoint to prevent agents from going off-track.
Plugins / Tools API
A standardized interface for agents to discover and use external tools. This allows a single agent to access capabilities from many third-party services (flight booking, restaurant ordering, APIs, etc.) without being retrained.
MCP (Model Context Protocol)
An emerging standard for how LLMs communicate with tools and hosts. Goal: create interoperability so any model can use any tool, moving away from proprietary implementations.

Types of Agents

Simple Agent
Handles a single task with one or two tools. Straightforward, low-risk, easy to debug. Example: Answering questions by searching the web.
Autonomous Agent
Operates independently for extended periods. Multiple tools, complex reasoning, long planning horizons. Requires robust guardrails and human checkpoints.
Coding Agent
Specialized for writing and debugging code. Tools include code execution, file access, version control. Examples: GitHub Copilot, Claude Code.
Multi-Agent System
Multiple specialized agents working together on complex problems. One agent might research, another analyzes, a third writes. Requires coordination.
