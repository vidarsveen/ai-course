Streamlit for Python UI

Building Web Interfaces with Pure Python
Streamlit is a Python library that turns data scripts into interactive web applications. Instead of writing HTML/CSS/JavaScript, you write pure Python. Streamlit handles all the UI rendering for you. Perfect for data dashboards, ML model demos, internal tools, and rapid prototypes where you need a web interface without learning web development.

import streamlit as st st.title("Greeting App") name = st.text_input( "Enter your name" ) if st.button("Greet"): st.write(f"Hello {name}")

Run with: streamlit run app.py

Getting Started with Streamlit
1. Install Streamlit
In your PyCharm terminal:
pip install streamlit
2. Write Python Code
Create app.py with Streamlit commands. No HTML/CSS needed:
st.title("My App")
st.write("Hello world")
value = st.slider("Pick a number", 0, 100)
3. Run It
From PyCharm terminal:
streamlit run app.py
Opens browser automatically with live preview
4. Iterate with Gemini
Ask Gemini to add features: "Add a file uploader to my Streamlit app"
Gemini generates the code, you paste it in
